# 分數學習教材

一個互動式的分數學習網頁應用程式，專為小朋友設計，透過視覺化的畫線切分與填色方式來學習分數概念。

## 專案展示

🔗 [線上展示 Demo](https://idben.github.io/fraction-02/)

## 功能特色

### 答題機制
- **10 題挑戰**：每回合隨機出 10 題，每題 10 分，滿分 100 分
- **多元題庫**：分母涵蓋 2~10，多種切分方式組合
- **即時回饋**：答對/答錯會有音效與角色立繪提示
- **錯誤示範**：答錯時會顯示正確的切割與填色方式
- **重試機制**：答錯後可重試，但重試答對不計分
- **滿分獎勵**：100 分時會有特別的勝利音效與過關立繪

### 互動操作
- **畫線切分**：點擊矩形邊緣的綠色圓點，連接對應點畫出分割線
- **區域填色**：點擊分割後的區域進行填色或取消填色
- **刪除線條**：點擊已繪製的線條即可刪除
- **重置畫布**：清除所有線條和填色，重新作答

### 視覺設計
- **半圓形點位**：與矩形邊緣完美融合的綠色半圓點
- **動態矩形**：高度根據分母自動調整，確保可整除
- **分數提示**：矩形中央顯示大字分數，方便參考
- **Bootstrap 風格按鈕**：美觀的互動按鈕設計
- **響應式設計（RWD）**：支援桌面與手機

## 操作說明

1. 📖 閱讀題目，例如「請把 1/3 塗上顏色」
2. ✏️ 點擊矩形邊緣的綠色圓點，連接對應的點畫出分割線
3. 🎨 點擊分割後的區域進行填色
4. ✅ 按「確認答案」檢查是否正確
5. 🔄 按「重置」可清除所有線條和填色重來

## 題庫說明

支援分母 2~10 的分數題目，每種分母有多種切分方式：

| 分母 | 切分方式 |
|------|----------|
| 2 | 橫切2等分、縱切2等分、2×2格 |
| 3 | 橫切3等分、縱切3等分 |
| 4 | 橫切4等分、縱切4等分、2×2格、4×2格 |
| 5 | 橫切5等分、縱切5等分 |
| 6 | 橫切6等分、縱切6等分、3×2格、2×3格 |
| 7 | 橫切7等分、縱切7等分 |
| 8 | 橫切8等分、4×2格、4×4格、8×2格等 |
| 9 | 橫切9等分、縱切9等分、3×3格等 |
| 10 | 橫切10等分、5×2格、2×5格 |

## 技術規格

### 技術要求
- ✅ 純 HTML/CSS/JavaScript，無框架依賴
- ✅ 使用 SVG 繪製矩形、分割線與填色區域
- ✅ 使用原生 `<dialog>` 元素實現對話框
- ✅ 使用 CSS 變數設計（可調整尺寸、顏色）
- ✅ 響應式設計，支援桌面與手機瀏覽

### CSS 變數
```css
:root {
  --rect-width: 800px;
  --rect-height: 400px;
  --point-diameter: 30px;
  --line-width: 3px;
  --fill-color: rgba(100, 149, 237, 0.3);
  --point-color: #4CAF50;
  --rect-stroke: #4CAF50;
}
```

### RWD 斷點
- **桌面**：800×400px（預設）
- **平板**：600×300px（≤900px）
- **手機**：自適應寬度，保持 2:1 比例（≤650px）

## 使用方式

直接用瀏覽器開啟 `index.html` 即可使用，或部署到任何靜態網頁伺服器。

```bash
# 複製專案
git clone https://github.com/idben/fraction-02.git
cd fraction-02

# 直接開啟 index.html 或使用本地伺服器
```

- Firefox
- Safari
