# 學習分數的教材製作
## 操作需求
* 畫面上有一個矩形，橫向的邊有 7 個點切成 8 等分；縱向的邊有 3 個點切成 4 等分。
* 點選上半部的橫邊的點，可以連到下半部橫邊的點，並畫出直線。
* 點選左半部的縱向邊的點，可以連到右半部縱向邊的點，並畫出直線。
* 畫出的直線可以把矩形切割。
* 點選切割區可以填色，已填色的再點一下會取消填色。
* 最後按按鈕會顯示填色區佔總矩形的幾分之幾。

## 程式需求
* 純 HTML/CSS/JS
* 不使用 JS 框架
* 不使用 CSS 框架
* 使用 Proxy，如果需要更新元素內容的話
* 使用 querySelector 與 querySelectorAll，不使用 getElement 系列語法，如果需要的話
* 使用 dialog 代替任何 alert 和 comfirm 的需求


## 視覺需求
* 按鈕使用 Bootstrap 的按鈕設計，並有 btn-primary、btn-danger、btn-warning 等顏色。
* 背景使用白色。
* 矩形邊上的點使用半圓形呈現，並與矩形的邊融為一體。
* dialog 在畫面正中間，需要有半透明黑色的 backdraop；手機版則貼齊上面。
* 答題機制區不管在什麼裝置都在畫面正中央。
* 手機上要剛好一個螢幕的寬高，不出現捲軸。
* 切割用的矩形寬度 800，高度基本上 400，可以動態調整，像是分母為奇數時可以整除的數字
* images 資料夾中的圖片要放在 dialog 與 backdrop 中，用 CSS 控制顯示與隱藏
* images 資料夾的圖片出現的位置要在畫面的左下角，像 RPG 的立繪一樣

## 答題機制
* 小朋友導向，所以使用的語句用小朋友的語言
* 一題十分，一共十題，分數出現在右上角
* 題目是把矩形畫線切分，並填色，讓有顏色的部份剛好是 1/8 之類的
* 答對與答錯要有對應的音效，在 sounds 資料夾中的 correct 和 wrong
* 答對與答錯要有對應的顯示，在 images 資料夾中的 happy 和 sad
* 全部答對會有 victory 音效與 pass 圖片
